리스트 항목 : ol, ul, dl

ol(Order List) : 순차적 목록, 블록성질
type 속성에는 A(알파벳 대문자), a(알파벳 소문자), I(로마숫자 대문자), i(로마숫자 소문자), 1(아라비아 숫자). 기본값 1(아라비아 숫자)
start 속성을 통해서 최초의 시작 숫자 또는 문자를 지정할 수 있음

입사지원 순서

  1. 1차 면접

  2. 2차 면접
  3. 최종 면접
  4. 건강검진
  5. 합격여부
    1. 합격통보
    2. OT

ul(Unorder List) : 비순차적 목록. 블록성질
list-style로 초기 리스트 표기를 결정할 수 있음(circle, disc, square, none). 기본은 disc
