video tag를 활용한 저장된 영상 가져오기

비디오 태그를 구성하는 속성들

source라는 태그를 활용하여 영상의 형식을 다양하게 가져올 수 있다.

  1. loop : 반복재생(영상이 끝나도 자동재생)
  2. muted : 음소거모드
  3. controls : 재생버튼(제어)
  4. autoplay : 브라우저가 로딩된 후 영상이 자동재생
  5. playsinline : IOS는 버전 호환성상 autoplay가 안될 수도 있어서 같이 사용

유투브 영상 가져오기

?rel=0&autoplay=1&loop=1;playlist=M_l1IjV-Ycw&mute=1;&controls=0; : 속성설정

playlist 값 = embed/"..."
